# 🏛️ Museia – Museum Ticket Booking & AI Assistant

Museia is a full-stack museum booking platform that helps users explore museums, book tickets, get AI-based personalized recommendations, and interact with an AI-powered museum chatbot.

## 🚀 Features

✨ Browse Museums
🎟️ Book Museum Tickets
🖼️ Explore Shows & Exhibitions
🤖 AI Chatbot Assistant
🎯 Personalized Museum Suggestions
👤 User Authentication
💳 Payment Integration

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Akash4421/Museia.git
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```

Run backend:

```bash
node server.js
```

---

## 🔐 Environment Variables

Create `.env` file:

```env
PORT=5000
GEMINI_API_KEY=your_api_key
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
```
